Codecademy Chapter Website

The goal of this project was twofold: to host resources for the Codecademy Chapter that I founded in February 2021, and to build a webpage that dynamically responds as the browser size adjusts. The design used responsive units like ems, rems and @media queries.

Musicademy

The inspiration behind "Musicademy", was Codecademy, the site in which much of my development trainings had been through. I wanted to envision a fictional company that had career paths, but instead of to become a full-stack developer, they were to learn music theory, composition, history, arranging, or even performance skills. Using Flexbox to design the layout, I was able to position elements to my desire, I learned how to create a 'navbar', and used a flexible display to showcase the 'employees'.

Personal Brand Style Guide

This project focused on CSS fundamentals such as CSS Grid, Flexbox, text styling, colors, typography, and more. Google fonts were used to achieve a specific aesthetic, and the design guide has font suggestions, a color palate, and even suggestions on how to format code.

Flex Box Exercize
